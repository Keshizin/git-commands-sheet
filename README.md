# Git commands

## Configuration

```
  $ git config --list
  $ git config --list --show-origin

  $ git config --[system | global | local] user.name "yourName"
  $ git config --[system | global | local] user.email "yourEmail"

  $ git config --[system | global | local] core.editor emacs
  $ git config --[system | global | local] core.editor "path/editor.exe"

  $ git config --[system | global | local] init.defaultbranch main
```
